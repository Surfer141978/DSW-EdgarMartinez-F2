Descripcion de la pagina

El sitio web se enfoca en enseñar el uso de CSS Grid, una tecnología de diseño web que permite crear diseños complejos y flexibles en páginas web. A través de tutoriales y ejemplos prácticos, se brinda una guía detallada sobre cómo utilizar esta herramienta de manera efectiva para mejorar el diseño y la estructura de sitios web..

Al aprender CSS Grid, podrás:

  • Crear diseños responsivos y adaptables a diferentes tamaños de pantalla y dispositivos.
  • Organizar y estructurar el contenido de tu sitio web de manera más eficiente y efectiva.
  • Ahorrar tiempo y esfuerzo al crear diseños complejos y de varias columnas.
  • Mejorar la accesibilidad y la usabilidad de tu sitio web al crear diseños más intuitivos y fáciles de usar.

Tema seleccionado CSS GRID

Definir un contenedor de cuadrícula

Lo primero que debe hacer para utilizar CSS Grid es definir un contenedor de cuadrícula. Esto se hace mediante la propiedad display: grid en el selector CSS que corresponde al contenedor. Por ejemplo:

                        .grid-container {
                            display: grid;
                            grid-template-columns: auto auto auto;
                        }
                        .grid-item {
                            padding: 20px;
                            font-size: 30px;
                            text-align: center;
                        }

Este código define un diseño de cuadrícula utilizando CSS Grid. La clase .grid-containerse utiliza para contener los elementos que se van a mostrar en la cuadrícula. Se define como una cuadrícula de tres columnas de ancho automático (se ajustará automáticamente al tamaño del contenido) mediante la propiedad grid-template-columns: auto auto auto;.

clase .grid-itemse utiliza para los elementos que se van a mostrar en la cuadrícula. Se definen algunas propiedades de estilo, como el relleno ( padding), el tamaño de fuente ( font-size) y la alineación de texto ( text-align).

Definir las filas y columnas de la cuadrícula

Una vez que haya definido el contenedor de cuadrícula, puede definir las filas y columnas de la cuadrícula. Para hacerlo, utiliza las propiedades grid-template-rowsy grid-template-columns. Por ejemplo:

                            .container {

                                display: grid;

                                grid-template-rows: 100px 200px;

                                grid-template-columns: 1fr 2fr 1fr;

                        }

En este ejemplo, la cuadrícula tiene dos filas, la primera de 100 píxeles de altura y la segunda de 200 píxeles de altura. También tiene tres columnas, la primera y la tercera ocupada un tercio del ancho disponible, mientras que la segunda ocupa dos tercios.

Columnas de cuadrícula

Las líneas verticales de los elementos de la cuadrícula se denominan columnas .

Filas de cuadrícula

Las líneas horizontales de los elementos de la cuadrícula se denominan filas .

Brechas de cuadrícula

Los espacios entre cada columna/fila se denominan espacios

Brechas de cuadrícula

Los espacios entre cada columna/fila se denominan espacios .

ejemplo column-gap

La propiedad column-gap establece el espacio entre las columnas:

    .grid-container {
        display: grid;
        column-gap: 50px;
        grid-template-columns: auto auto auto;
      }

La propiedad "column-gap" define el espacio entre las columnas de una cuadrícula y especifica la distancia horizontal entre elementos adyacentes. En este caso, la propiedad se establece en 50px, lo que significa que cada columna tendrá un espacio de 50px entre ellas.

ejemplo row-gap

La row-gap propiedad establece el espacio entre las filas: 

    .grid-container {
        display: grid;
        row-gap: 50px;
        grid-template-columns: auto auto auto;
      }

tiene la propiedad display establecida en grid, lo que la convierte en un contenedor de cuadrícula. Luego, la propiedad row-gap se establece en 50px, lo que significa que habrá un espacio de 50px entre cada fila en la cuadrícula.

Ejemplo gap

La gap propiedad es una propiedad abreviada para el row-gap y las column-gap propiedades: 

    .grid-container {
        display: grid;
        gap: 50px 100px;
        grid-template-columns: auto auto auto;
      }

Este código establece una cuadrícula utilizando CSS Grid, donde los elementos de la cuadrícula tendrán un espacio de 50 píxeles entre las filas y un espacio de 100 píxeles entre las columnas. Es decir, se establece una separación diferente entre filas y columnas usando la propiedad "gap".

Contenedor de cuadrícula CSS

La propiedad grid-template-columns

La grid-template-columnspropiedad define el número de columnas en su diseño de cuadrícula y puede definir el ancho de cada columna. El valor es una lista separada por espacios, donde cada valor define el ancho de la columna respectiva. Si desea que su diseño de cuadrícula contenga 4 columnas, especifique el ancho de las 4 columnas o "automático" si todas las columnas deben tener el mismo ancho.

                        .grid-container {
                            display: grid;
                            grid-template-columns: auto auto auto auto;
                          }

La grid-template-columnspropiedad también se puede utilizar para especificar el tamaño (ancho) de las columnas.

Este código define una cuadrícula (grid) con 4 columnas del mismo ancho. El valor "auto" indica que el ancho de cada columna se ajustará automáticamente en función del contenido que tenga.

La propiedad grid-template-rows

La grid-template-rowspropiedad define la altura de cada fila

                            .grid-container {
                                display: grid;
                                grid-template-rows: 80px 200px;
                              }

El valor es una lista separada por espacios, donde cada valor define la altura de la fila respectiva, en este caso, se están definiendo dos filas con una altura de 80 píxeles para la primera fila y 200 píxeles para la segunda fila.

Ejemplos de la propiedad de justificar el contenido

La justify-contentpropiedad se utiliza para alinear toda la cuadrícula dentro del contenedor.

space-evenly

Utilice la propiedad de justificar el contenido para alinear la cuadrícula dentro del contenedor. El valor "space-evenly" dará a las columnas la misma cantidad de espacio entre ellas y alrededor de ellas:

    .grid-container {
        display: grid;
        justify-content: space-evenly;
      }

space-between

Utilice la propiedad de justificar el contenido para alinear la cuadrícula dentro del contenedor. El valor "space-between" dará a las columnas la misma cantidad de espacio entre ellas:

    .grid-container {
        display: grid;
        justify-content: space-between;
      }

center

Utilice la propiedad de justificar el contenido para alinear la cuadrícula dentro del contenedor. El valor "center" alineará la cuadrícula en el medio del contenedor:

    .grid-container {
        display: grid;
        justify-content: center;
      }

start

Utilice la propiedad de justificar el contenido para alinear la cuadrícula dentro del contenedor. El valor "start" alineará la cuadrícula al comienzo del contenedor:

    .grid-container {
        display: grid;
        justify-content: start;
      }

end

Utilice la propiedad de justificar el contenido para alinear la cuadrícula dentro del contenedor. El valor "end" alineará la cuadrícula al final del contenedor:

    .grid-container {
        display: grid;
        justify-content: end;
      }
